# Streamer Frontend

## Environment Variables

To run this project, you will need rename `.env.example` file to `.env`

## Run Locally

Clone the project

```bash
  git clone https://github.com/VoloshynV/streamer-front
```

Go to the project directory

```bash
  cd streamer-front
```

Install dependencies

```bash
  yarn install
```

Build App

```bash
  yarn build
```

Start the server

```bash
  yarn start
```
